## AUTOMATIC INSTALLATION (arch)
<div/>




- Download the [installer](https://github.com/maxlttr/wm/blob/main/installer.sh) 
```sh
https://github.com/maxlttr/wm/blob/main/installer.sh
```
- Make the installer executable:
```sh
cd Downloads
chmod +x installer.sh
```
- Run it !
```sh
./installer.sh
```
